- #concept [[Python]] Variables
- Definition
  id:: 69531fe2-2583-4c65-8667-50979a1e710d
	- A variable in Python is a name bound to an object in memory.
- Examples
	- ```python
	  x = 10
	  name = "Solomon"
	  is_active = True
	  ```
- Key properties
	- Dynamically typed: the type is associated with the object, not the name.
	- Names are just labels pointing to objects.
